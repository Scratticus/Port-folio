About Me

I'm Adam, but my friends call me Scrat and you can too! I am excited to be writing code! My industrial experience has often involved code and coding, with my role requiring me to write specifications for machine code, designing my own stock tracking programs and coding myself in SQL and Visual Basic. I am advanced at deep datamining with SQL, displaying my findings with vba integration creating applications within programs like excel to call SQL code and populate graphs for analysis.

For the last few years I have been building a community online as a Twitch stream content creator. The Scratticus Academy taught interested players how to play Dungeons and Dragons and other Tabletop Roleplaying games live on stream. I learnt LOADS about streaming and have offered consultations to diagnose and repair laggy stream set ups. I also learnt a fair amount of HTML, CSS and JS as I built my own web elements to be displayed on stream. Whilst the stream was a huge success it is time to separate my working life and hobbies once more. I have enjoyed code and I am keen to discover more about myself and coding language!
